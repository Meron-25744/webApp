## 💻 自分のローカル環境にGitHubのファイルをクローンして、動くようにする

https://qiita.com/takeru-hirai/items/4fbe6593d42f9a844b1c
このサイトに従って、gitをダウンロードし、ssh鍵？まで作る

### 2️⃣ リポジトリのクローン
その後、windowsのターミナルを開き、自分の好きなディレクトリまで移動し、
git clone git hubでcode->urlコピーしたものを貼る
このコマンドを打つ

### 3️⃣ requirementsのダウンロード
その後、自分のエディターで↑で作ったディレクトリまで移動し、
pip install -r requirements.txt　を入力
（vs code なら　vs code->ターミナル->コマンドプロンプトで入力）

### 4️⃣ データベースに反映させる
最後に3と同じ場所で　python manage.py migrate を入力

## 💻 自分でファイル(html)の中身を変更して、保存する場合　ブランチ名はここではdevelopです

### 1️⃣ ブランチの更新
git fetch origin で最新のブランチ（ファイルみたいな）を更新する

### 2️⃣ 自分がいるブランチの確認
git branch -r　でorigin/ブランチ名があるか確認（赤字で出る）エラーじゃないよ

### 3️⃣ ブランチの移動
次にgit checkout 移動先のブランチ名　で移動する

### 4️⃣ 自分がいるブランチの確認
git branchで自分がdevelopにいることを確認して
